# SQLite / WebAssembly Client test

## Install

```
git clone https://github.com/ICJIA/sqlite-wasm.git
cd sqlite-wasm
npm install
cp node_modules/sql.js/dist/sql-wasm.js .
cp node_modules/sql.js/dist/sql-wasm.wasm .
```

## Initialize database

```
node init.js
```

## Start client

```
npm run serve
```

## View demo

```
http://localhost:5000
```
